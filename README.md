## 📱 Déploiement

Ce portfolio est automatiquement déployé via GitHub Pages :

**URL** : https://eliottvanovermeir-sudo.github.io/portfolio/

### Configuration GitHub Pages

1. Accédez aux **Settings** du repository
2. Allez dans **Pages**
3. Sélectionnez **Deploy from a branch**
4. Choisissez **main** branch
5. Enregistrez
